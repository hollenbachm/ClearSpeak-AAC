# ClearSpeak

ClearSpeak is a simple AAC (Augmentative and Alternative Communication) tool designed to help stroke survivors and people with speech impairments communicate basic needs and phrases.

The project was inspired by my father's stroke in October 2025 and the communication challenges that followed.

## Features

• Quick Needs board  
• Sentence builder with text-to-speech  
• Easy Mode (tap to speak)  
• Custom caregiver cards  
• My People / My Routine shortcuts  
• Voice speed and pitch controls  
• Local storage for privacy
